Footfall Counter: A Video-based People Counting System

Overview:

Our project aims to develop a robust and efficient program for accurately counting the number of unique individuals in a video. The system detects and tracks individuals throughout the video, assigning a unique identifier to each person to ensure accurate counting.

Implementation:

We implemented the footfall counting system using Python along with the OpenCV library for video processing and computer vision tasks. The program accepts a video file as input and employs advanced object detection and tracking techniques to identify individuals appearing in the video. We utilized deep learning-based models for person detection and tracking, which allow for accurate identification and tracking.

Key Features:

  • Accepts video files of various formats as input.
  • Utilizes state-of-the-art object detection and tracking with YOLOv8 model.
  • Assigns a unique identifier to each individual for accurate counting.
  • Provides clear and comprehensive instructions for setting up and running the program, including installation of dependencies.
  • Creates a log of each frame to identify each frame with date, time, frame number, persons tracked in and out time, daily hours per person, total individuals detected, and total people inside.

How it Works:

  1. The program reads the input video and processes each frame sequentially by yielding frames.
  2. Each frame is resized and analyzed using the YOLO person detection model to identify individuals.
  3. Detected individuals are tracked across frames and assigned to each individual to ensure accurate counting.
  4. The total in and out footfall count, daily hours per person, and total people inside all available insights displayed as the output in a page using Flask.

Conclusion:

Our footfall counting system provides an effective solution for accurately counting the number of unique individuals in a video. The implementation showcases the capabilities of modern computer vision techniques and deep learning models in solving real-world problems efficiently.
